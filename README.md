# Shell Configuration

### Install homebrew

    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

### Install [yadm](https://github.com/TheLocehiliosan/yadm)

    brew install yadm
    yadm clone git@github.com:outaTiME/dotfiles.git
    brew install gpg
    yadm decrypt
